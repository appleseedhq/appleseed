Description:
This demo allows the user to load and parse a specified XML document.  Then, through a text box 
the user can navigate and manipulate the tree.  This is both a useful example of how to walk the 
XML tree using the XML Object Model and a useful tool for learning the XML Object Model.

More Details:
This comes with a sample file for use on this website.  If you run the demo on your site, however, you can access other files.

Compatibility/Platform Compatibility:
You must be running Microsoft Internet Explorer 5 or greater on Win32 or Unix platforms to view or host this demo.
