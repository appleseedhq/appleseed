socket_base::shutdown_type

Different ways a socket may be shutdown. 

enum shutdown_type
Values

shutdown_receive

Shutdown the receive side of the socket.

shutdown_send

Shutdown the send side of the socket.

shutdown_both

Shutdown both send and receive on the socket.
