Description

In all cases, proto::functional::unpack_expr<Tag, Domain>()(seq) is equivalent to proto::unpack_expr()<Tag, Domain>(seq).

proto::functional::unpack_expr<Tag>()(seq) is equivalent to proto::unpack_expr()<Tag>(seq).
